# agusys-LINE

## LINEbot 版あぐしす

{: align="center"}
![logo](public/images/logo-s.png "agusys")

### 用途

- 被災状況の報告
- 復旧後の確認

### できること

- 位置情報の送信・スプレッドシートへの記録
- 写真/動画の撮影・Google ドライブへの保存
- テキストによるスプレッドシートへの記録
